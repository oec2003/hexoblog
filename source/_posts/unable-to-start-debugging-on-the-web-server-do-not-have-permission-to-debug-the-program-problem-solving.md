最近维护以前用03做的一个项目,又重新将03装上了，很久都没有用03了,装上后打开项目没有问题,但按F5运行却出现下面错误
<!--more-->

![2010-12-31_152927](http://oec2003.qiniudn.com/2010-12-31_152927.gif)

解决方法如下:

在INTERNET选项–>安全–>选择internet–>点击“自定义级别”–>移到“用户验证”，选择“自动使用当前名和密码登录”，点击“确定”。
